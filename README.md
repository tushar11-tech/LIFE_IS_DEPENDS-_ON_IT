# 🚀 My First Data Analytics Project: Sales Performance Dashboard

Welcome to my portfolio! This repository marks the beginning of my journey as a Data Analyst. This interactive dashboard tracks $127K in total revenue and transforms raw transaction logs into clear retail metrics.

### 📋 The Problem
Raw sales logs lacked clear visibility into which specific products and territories were driving the actual revenue.

### 💡 The Solution
Cleaned transaction records, built seasonal trend charts, and created high-level KPI cards for instant performance tracking.

### 🔍 Key Insights Discovered
* **Volume vs. Value:** Mobile phones sold the highest total volume of units, but Cameras generated the highest total overall revenue.
* **Top Market:** Vadodara was the top-performing regional market.
* **Seasonality:** February was the peak sales month.

### 🛠️ Interactive Feature
Added an Employee ID filter so managers can instantly drill down into individual sales team performance.
